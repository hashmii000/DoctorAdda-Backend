📚 API Documentation Overview

1. Auth APIs
POST /api/auth/register - Register user with OTP
POST /api/auth/login - Login with OTP
POST /api/auth/verify - Verify OTP
POST /api/auth/resend-otp - Resend OTP



2. User APIs
GET /api/user/profile - Get user profile
PUT /api/user/update - Update profile and settings



3. Doctor APIs
POST /api/doctor/create - Register doctor
GET /api/doctor/list - Get list of doctors
PUT /api/doctor/update/:id - Update doctor profile
POST /api/doctor/set-availability - Set available slots


4. Hospital APIs
POST /api/hospital/create - Add hospital
GET /api/hospital/list - List all hospitals
GET /api/hospital/:id - Get details of a hospital
PUT /api/hospital/update/:id - Update hospital info


5. Booking APIs
POST /api/booking/create - Book appointments
GET /api/booking/user/:id - View user bookings
PUT /api/booking/cancel/:id - Cancel a booking
